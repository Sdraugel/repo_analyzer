Holidays.py

Description:

This Python script is designed to calculate the dates of certain holidays in a given year and return a label for a date if it is a holiday. The script starts with a header that includes legal information about the software's development and export restrictions. The script then defines several functions: - `MonthStart(monthStartStr)`: This function takes a string in the format 'Jan 1 2003' and returns the day of the week and the day of the year for the start of the month. - `ThirdMonday(monthStartDOW, monthStartDOY)`: This function calculates the day of the year for holidays that fall on the third Monday of a month. - `calculateHolidayList(yearStr)`: This function calculates the dates of several holidays in a given year, including New Year's Day, Martin Luther King Jr. Day, Washington's Birthday, Memorial Day, Independence Day, Labor Day, Columbus Day, Veterans Day, Thanksgiving Day, and Christmas Day. The dates are added to a list called `Holidays`. - `getHolidayLabel(startTime)`: This function takes a date and checks if it is a holiday by comparing it to the dates in the `Holidays` list. If the date is a holiday, the function returns the label of the holiday. If the date is not a holiday, the function returns an empty string. The script also includes a static member `_holidayList` that serves as a cache. This is used in the `getHolidayLabel(startTime)` function to store the list of holidays for a given year, so that the list does not need to be recalculated every time the function is called.
