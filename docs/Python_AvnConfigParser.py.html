AvnConfigParser.py

Description:

This Python code defines two classes: `MultiOrderedDict` and `AvnConfigParser`. `MultiOrderedDict` is a subclass of `collections.OrderedDict`. It overrides the `__setitem__` method to join values for duplicate keys into a comma-separated list instead of overwriting them. This is useful when you want to keep all values associated with a key in a dictionary-like data structure. The `keys` method is also overridden to return the keys of the dictionary. `AvnConfigParser` is a subclass of `configparser.ConfigParser`. It's designed to merge duplicate keys into a comma-separated list to better work with Java code that uses `org.apache.commons.configuration`. It also defaults to better match Python 2.7. This class overrides the `get` method to replace '""' or "''" with an empty string. The code also contains a software history log and a legal notice from Raytheon Company, the developer of the software. The notice includes information about export control restrictions, the contractor's name and address, and a reference to a master rights file for licensing information.
