Description:

This Python script is part of a software developed by Raytheon Company under a contract with the US Government. The software contains export-restricted data, and its export or transfer is controlled by U.S. law. The script defines a class `TextProduct` that inherits from a base class `CivilEmerg.TextProduct`. This class is used to produce a "Shelter In Place Warning" text product. The class contains a dictionary `Definition` that holds various attributes of the product, such as its display name, output file, default edit areas, product name, WMO code, product pil, and IDs for storing to AWIPS text database and for transmitting to AWIPS WAN. The `TextProduct` class has an initializer method `__init__` that calls the initializer of the base class `CivilEmerg.TextProduct`. It also has a method `_makeProduct` that takes in a forecast `fcst`, an edit area `editArea`, an area label `areaLabel`, and a dictionary of arguments `argDict`. This method appends a message to the forecast and returns it. The script also imports several modules, including `CivilEmerg`, `TextRules`, `string`, `time`, `re`, `os`, `types`, and `copy`.
