nwpsCG1MFR.py

Description:

This Python script is part of a software developed or modified by Raytheon Company under a US Government contract. The software contains export-restricted data, and its dissemination requires an export license or other authorization. The script defines a module that calculates surface weather elements from the nwpsCG1 model output. This is achieved by defining a class `nwpsCG1MFRForecaster` that inherits from the `nwpsCG1Forecaster` class. The `nwpsCG1MFRForecaster` class has a constructor that initializes its parent class with the arguments "nwpsCG1MFR" and "nwpsCG1MFR". The script also defines a `main` function that creates an instance of the `nwpsCG1MFRForecaster` class, runs it, and then notifies the GFE with 'MFR'. The `main` function is executed if the script is run as the main program. The script imports two modules, `Init` and `nwpsCG1`. It's also noted that this is a base file and is not intended to be overridden, but it can be subclassed to override behavior. This is part of a larger system, as it references a Master Rights File for further licensing information and a GFE Online Help for guidance on creating a new smart init.
