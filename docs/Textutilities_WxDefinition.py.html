WxDefinition.py

Description:

This is a Python script that wraps a Java object, specifically a `WxDefinition` object. The script was developed or modified by Raytheon Company under a US Government contract and contains export-restricted data. The `WxDefinition` class has an initializer that takes a `javaDef` object. This object is stored as a private attribute `__def`. The class has several methods that call methods on the `__def` object: - `javaDefinition()`: returns the `__def` object. - `typeDesc(wxType)`: returns the type description of a weather type. - `intensityDesc(wxType, inten)`: returns the intensity description of a weather type and intensity. - `attributeDesc(wxType, attr)`: returns the attribute description of a weather type and attribute. - `coverageDesc(wxType, cov)`: returns the coverage description of a weather type and coverage. - `visibilityDesc(wxType, vis)`: returns the visibility description of a weather type and visibility. The script also mentions that it is an absolute override file, meaning that a higher priority version of the file will completely replace a lower priority version of the file.
