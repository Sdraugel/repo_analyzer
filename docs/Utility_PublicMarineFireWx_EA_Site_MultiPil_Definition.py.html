PublicMarineFireWx_EA_Site_MultiPil_Definition.py

Description:

This code is a configuration file for a software developed by Raytheon Company. It contains information about the company, the export restrictions on the software, and the licensing information. The code then defines a base file that is not intended to be overridden. It also states that the software is in the public domain, provided "as is", without any technical support or warranty. The main purpose of this file is to generate a combinations file that contains all public and marine codes for a given site. This is similar to a product definition file, but only contains what is needed for the installation to generate a combinations file. The file defines a dictionary named `Definition`. This dictionary is used to store configuration settings for the software. The `defaultEditAreas` key in the `Definition` dictionary is used to store the name of the generated combinations file. The `mapNameForCombinations` key in the `Definition` dictionary is used to store the map background for creating the combinations file. It is a list that contains the names of the zones for a given site.
