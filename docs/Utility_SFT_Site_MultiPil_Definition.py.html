SFT_Site_MultiPil_Definition.py

Description:

The code is a configuration file for a software developed by Raytheon Company. The software is related to the AWIPS II system, which is used for weather forecasting and data analysis. The first part of the file contains legal information about the software, including the fact that it contains export-restricted data, and the contractor details. The second part of the file is a disclaimer stating that the software is in the public domain, provided "as is", without any warranty or technical support. The third part of the file is the actual configuration for a specific part of the software, the SFT formatter for a site. The configuration is stored in a Python dictionary named `Definition`. The dictionary contains several keys, each representing a different configuration item. Some of these items are required, such as the display name and default edit areas, while others are optional, such as the database and debug flags. The configuration items include details about the station, the product, the zone code, the state name, the location of the WFO (Weather Forecast Office), and the product IDs for storing to the AWIPS text database and for transmitting to the AWIPS WAN (Wide Area Network). There's also a configuration for the output file path. The configuration also includes optional items such as the source database, debug flag, and flags for including PoP (Probability of Precipitation) and QPF (Quantitative Precipitation Forecast) in the output. These optional items are currently commented out.
