UserInfo.py

Description:

This Python script defines a class called `UserInfo` that inherits from two other classes: `TextRules` and `SampleAnalysis`. The purpose of this class is to get the name of the forecaster, which is then added to the end of a product. The `UserInfo` class has two methods: `forecasterDict` and `_getForecasterName`. The `forecasterDict` method returns a dictionary where the keys are usernames and the values are the corresponding forecaster names. The `_getForecasterName` method takes an argument dictionary (`argDict`). It tries to get the debug level from the `argDict`. If it's not available, it sets the debug level to 1. It then gets the current user's name from the environment variable `USER`. If this username is in the forecaster dictionary, it retrieves the corresponding forecaster name. If not, it sets the forecaster name to "NATIONAL HURRICANE CENTER". The method then prints the forecaster name for debugging purposes and returns it. The script also imports three modules: `TextRules`, `SampleAnalysis`, and `os`. The `os` module is used to get the value of the environment variable `USER`.
