Description:

This Python script is part of a software developed or modified by NOAA/NWS/OCP/ASDT. It is a base file that is not intended to be overridden. The software is in the public domain, furnished "as is", without technical support, and with no warranty, express or implied, as to its usefulness for any purpose. The script is for generating a Blue Alert message, a type of civil emergency alert. The output file is stored in the directory "{prddir}/TEXT/BLU_.txt". It also contains some configurations such as the display name, product name, WMO code, product pil, product ID for storing to AWIPS text database, and product ID for transmitting to AWIPS WAN. The script imports several modules: CivilEmerg, TextRules, string, time, re, os, types, and copy. It then defines a class `TextProduct` that inherits from `CivilEmerg.TextProduct`. A deep copy of the `CivilEmerg.TextProduct.Definition` dictionary is made and several key-value pairs are added or modified in the `Definition` dictionary. The class `TextProduct` has two methods: `__init__` and `_makeProduct`. The `__init__` method initializes the class by calling the `__init__` method of the parent class. The `_makeProduct` method takes four parameters (fcst, editArea, areaLabel, argDict) and appends a message to the `fcst` string. The message is "The following message is transmitted at the request of the " followed by the value of `self._source`. The method then returns the updated `fcst` string.
