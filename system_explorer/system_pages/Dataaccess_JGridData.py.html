Description:

This Python script is part of a larger software package developed by Raytheon Company under a US Government contract. The script is subject to US export control laws due to the presence of export-restricted data. The script implements a class called `JGridData` that wraps around a Java `IGridData` object. This class is used to handle grid data in the software package. The `JGridData` class has several methods for accessing specific properties of the grid data: - `getParameter`: Returns the parameter of the grid data. - `getUnit`: Returns the unit of the grid data. - `getRawData`: Returns the raw data of the grid data. If a unit is specified, the data is converted to that unit before being returned. - `getLatLonCoords`: Returns the latitude and longitude coordinates of the grid data. The class also overrides the `__getitem__` method to allow accessing these properties using dictionary-like syntax.
