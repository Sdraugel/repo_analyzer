7724_remove_comms_jmsconnection_element.py

Description:

This script is designed to update specific XML files on a server, specifically the files 'comms.xml' and 'comms-practice.xml' located in the '/awips2/bmh/conf/' directory. The update involves removing the 'jmsConnection' element from these XML files. Here's a step-by-step breakdown of what the script does: 1. It defines a list of paths to the XML files it will update. 2. The `rewrite_comms_xml` function is defined to handle the updating process for a single file. It first checks if the file exists. If not, it prints a message and skips the file. 3. If the file does exist, it opens the file and parses the XML into an ElementTree object, which allows for easy manipulation of the XML elements. 4. It then searches for all 'jmsConnection' elements in the root of the XML document. If no such elements are found, it prints a message and returns. 5. If 'jmsConnection' elements are found, it creates a backup of the current file, appending the current date to the filename. 6. It then removes all 'jmsConnection' elements from the XML document and writes the updated XML back to the original file. 7. The `main` function loops over all the paths defined at the start of the script, calling `rewrite_comms_xml` for each one. If an exception occurs during the update of a file, it prints an error message and the traceback of the exception. 8. Finally, if the script is run as the main module, it calls the `main` function to start the update process.
