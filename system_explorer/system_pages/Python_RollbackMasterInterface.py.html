RollbackMasterInterface.py

Description:

The code is a Python script that defines a class `RollbackMasterInterface` which extends from the `MasterInterface` class. This class is designed to act as a controller for inspecting and running Python scripts from Java. It differs from the base `MasterInterface` class because it utilizes a rollback importer. Here's a breakdown of the class and its methods: - `__init__(self, scriptPath)`: This is the constructor of the class. It initializes an instance of the class and sets the `_scriptPath` attribute to the path provided as an argument. - `importModules(self)`: This method is used to import modules from the script path. - `getStartupErrors(self)`: This method returns a list of import errors that occurred during the startup of the program. It uses the `ArrayList` class from `java.util` package to store the errors. - `addModule(self, moduleName)`: This method adds a module to the list of modules and then reloads all modules. - `reloadModule(self, moduleName)`: This method reloads a specific module and then reloads all modules. - `removeModule(self, moduleName)`: This method removes a specific module from the list of modules and then reloads all modules. - `reloadModules(self)`: This method reloads all modules.
