DGT_Site_MultiPil_Definition.py

Description:

This code is a configuration file for a Drought Monitor (DGT) product for a specific site. It sets up all the product definition overrides for the DGT formatter for a given site. The code begins by creating an empty dictionary named `Definition`. This dictionary is then populated with key-value pairs that define various configuration settings for the DGT product. The configuration settings include: - `displayName`: The name of the DGT product. - `fullStationID`: The full station identifier. - `wmoID`: The World Meteorological Organization ID. - `pil`: The product pil. - `stateName`: The name of the state. - `wfoCityState`: The location of the Weather Forecast Office (WFO). - `textdbPil`: The product ID for storing to the AWIPS text database. - `awipsWANPil`: The product ID for transmitting to the AWIPS WAN. - `outputFile`: The path and name of the output file. - `areaType` and `mapNameForCombinations`: These are optional and can be uncommented if issuing for zones. - `localWebsites`: A list of local websites related to the DGT product. - `acknowledgements`: A string containing acknowledgements to local, regional, or national agencies. - `addressWFO`: The address of the local WFO. There are also optional configuration items like `debug` and `purgeTime` that can be uncommented and set as needed. The code also mentions that there are additional definition settings that can be set in the `DGT.py` TextProduct.
