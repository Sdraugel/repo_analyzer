MockGridData.py

Description:

This part of the file is a Python script that contains a class definition for a mock object, `MockGridData`. This mock object is intended to act as a placeholder for a Java `IGridData` object during unit testing of `HazardUtils.py`. The `MockGridData` class has a single attribute, `slice`, and a single method, `getGridSlice()`. The `slice` attribute can be set when an instance of `MockGridData` is created (via the constructor) or can be manipulated directly. The `getGridSlice()` method simply returns the current value of the `slice` attribute. The script also includes a significant amount of metadata and legal information about the software, including the developer (Raytheon Company), the contract under which it was developed, and export control restrictions. It also provides contact information for the contractor and references a master rights file for further licensing information.
