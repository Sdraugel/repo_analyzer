PublicMarine_EA_Site_MultiPil_Definition.py

Description:

This code is a Python script that is part of a larger software developed by Raytheon Company. It contains important legal information about the software, including export restrictions and licensing details. The code is a configuration file that is used to generate a combinations file containing all public and marine codes for a specific site. It is similar to a product definition file, but only contains what is needed for the installation to generate a combinations file. The script defines a dictionary called `Definition` and sets two keys within it: 1. `defaultEditAreas`: This key is assigned the name of the generated combinations file. The name of the file will be "EditAreas_PublicMarine__", where `` and `` are placeholders that will be replaced with the actual site and MultiPil values. 2. `mapNameForCombinations`: This key is assigned a list of two strings, "Zones_" and "Marine_Zones_", which are the names of the map backgrounds used for creating the combinations file. Again, `` is a placeholder that will be replaced with the actual site value. This code is in the public domain, provided as is, without technical support or any warranty.
