SendGridsToNDFD.py

Description:

This Python script is a command-line utility for sending a site's grid data to the National Digital Forecast Database (NDFD). The script begins with a large comment block providing information about the software's origin, export control, and licensing. It also includes a software history log, detailing changes made over time. The script then imports necessary modules and classes, including `argparse` for command-line argument parsing, `ExportGridsRequest` for creating the request to export the grids, and `ThriftClient` and `UsageArgumentParser` from the `awips` package. The `SendGridsToNDFD` class is defined with a `send` method. This method takes a site, host, and port as arguments. It creates a Thrift client and an `ExportGridsRequest`, sets the site and mode of the request, and sends the request. If the request is successful, it returns 0; otherwise, it prints an error message and returns 1. The `processArgs` function is used to parse command-line arguments. It sets up an argument parser and defines three required arguments: site, host, and port. It then parses the arguments and returns them in a namespace. The `main` function processes the command-line arguments, creates an instance of `SendGridsToNDFD`, and calls the `send` method with the parsed arguments. Finally, if the script is run as a standalone program (not imported as a module), it calls the `main` function.
