Description:

This Python script is a part of a software developed or modified by Raytheon Company under a contract with the US Government. It contains export-restricted data and its dissemination to non-U.S. persons requires an export license or other authorization. The script is a base file that is not intended to be overridden. However, it can be subclassed to override behavior. It is related to the Smart Initialization Configuration section of the GFE Online Help. The script was updated by Joe Maloney on 2016-07-18 for the nwpsCG1MFL model. The script imports everything from two modules: Init and nwpsCG1. The script defines a class named 'nwpsCG1MFLForecaster' which is a subclass of 'nwpsCG1Forecaster' from the nwpsCG1 module. The class has an '__init__' method that initializes the class with the model name and identifier as "nwpsCG1MFL". The script also defines a 'main' function which creates an instance of 'nwpsCG1MFLForecaster', runs the forecaster, and then notifies GFE with the identifier 'MFL'. Finally, the script checks if it is being run directly (not imported as a module), and if so, it calls the 'main' function.
