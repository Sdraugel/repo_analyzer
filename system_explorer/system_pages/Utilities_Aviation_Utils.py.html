The `Aviation_Utils` class has an initializer method `__init__` that takes two parameters: `dbss` and `eaMgr` (with a default value of `None`). This method calls the initializer of the parent class with `dbss` as the argument. The class also has a method `getModelWeights` that takes two parameters: `dialogNames` and `varDict`. This method is used to get the model weights from the dialog names and variable dictionary. The method initializes an empty list `models` and an empty dictionary `modelWeights`. Finally, it returns the `models` list and the `modelWeights` dictionary.
